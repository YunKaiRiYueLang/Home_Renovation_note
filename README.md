# Home_Renovation_note
房屋装修笔记，杭州

四月初在杭州萧山买房，5月份决定装修。简单了解后明显感觉市场鱼龙混杂，有的人说话甚至狗屁不通却想要我交定金甚至签合同，真的感觉无语。今晚决定将搜集的资料整理成笔记，以被自己查看。

为了避免沟通中的名词混淆，必须明白下面的名词，及其背后的含义。

全包：指的是装修公司包工包料。人工和材料的采购、施工与管理都由装修公司负责。注意 **全包在装修公司眼里只是的形容词，不是名词更不是定义，全包所包含的内容都需要自己查看**，有可能缺乏某一部分，这部分花费的资金将以万为单位更亮。

半包：装修公司负责人工 + 辅材，业主自己购买主材。但是据我了解。**半包也是个形容词**，不同人/公司提供的半包，也不同，比如有的会包拆旧，有的包瓷砖等等。当然，有的这两项都没有。
